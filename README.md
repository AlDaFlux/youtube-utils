# confirmation-window-bundle

Have a confirmation window before link



## aldaflux_confirmation_window.yaml

```

aldaflux_confirmation_window:
    template: bootstrap4
    delete: true 
    customs:
        modify:
            title: "*************?"
            selector: ".modif"
            class: "warning"
        modify2:
            title: "*************?"
            selector: ".modif2"
            class: "danger"
    alerts:
        alert:
            selector: ".message"
            title: "*************?"
            button: " Go !! Go !! Go"
    
```
